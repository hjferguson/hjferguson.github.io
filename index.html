Introduction to Machine Learning

For this project, I wanted to make my own dataset and then train a language model. I went with Reddit because of their free API use. After generating a dataframe with thousands of subreddit titles and their corresponding subreddits as their labels, I cleaned and vectorized it in preparation for training. Using Scikit's Bayes model, I trained and tuned the data to an average F1 score of 80%. From there, I developed a webapp for users to enter a title, and have ML predict where that title would belong on Reddit.

Techstack: Python (Django/Scikit-Learn/Pandas), Typescript, React.js

Birthday Bot

This was my first project using APIs, and before I knew what a .csv was... It's definately in need of a V2, but I'm proud nontheless. The program has a list of friends with their phone numbers and birthdays. I have the program running daily on a raspberry pi, and cron jobs. The program checks the system date, and compares it to dates in the list. If there is a match, the program sends a text message to the recipient using SMTP. I went with this approach because I didn't want to pay for a service like Twilio.

Techstack: Python
